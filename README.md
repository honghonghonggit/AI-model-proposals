# AI Model Proposals

A collection of AI/neural network model proposals written 
during an undergraduate AI course, exploring novel 
combinations of classical neural network architectures.

> 📌 Note: If GitHub fails to preview the PDFs inline, 
> please click the download links below to view the full papers.

## Projects

### AUI — Adversarial Uncertainty Injection
A framework for designing robust rule-based expert systems 
against incomplete inputs.

### DNAM — Dreaming Neural Associative Memory
A self-reorganizing memory framework combining Hopfield 
networks, Hebbian learning, Kohonen competitive learning, 
and Bidirectional Associative Memory (BAM).

![DNAM Concept](<./DNAM (Dreaming Neural Associative Memory)/DNAM.png>)

📄 [Download Full Paper (DNAM.PDF)](https://github.com/honghonghonggit/ai-model-proposals/raw/main/DNAM%20(Dreaming%20Neural%20Associative%20Memory)/Dreaming%20Neural%20Associative%20Memory_A%20Self%20Reorganizing%20Memory%20Framework%20via%20Competitive%20Hebbian%20Replay.pdf)
